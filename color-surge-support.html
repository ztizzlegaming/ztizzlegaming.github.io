Color Surge: Logic Game – Support

About the App

Color Surge is a logic puzzle game inspired by graph coloring — a classic problem in graph theory. Each level challenges you to color connected stations while following simple rules that become increasingly difficult as the puzzles grow.

How to Play

  • Each station represents a point in a network.
  • Connections between stations represent constraints.
  • Rule: No two connected stations may share the same color.
  • Use logic, planning, and optional power-ups to solve each puzzle.
  • Earn stars and coins by completing levels efficiently.
